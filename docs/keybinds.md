# Keybinds

## Rofi

- `SUPER + SPACE`: launcher with `Apps`, `Files` and `Commands`.
- `SUPER + SHIFT + X`: command runner for shell commands.
- `SUPER + X`: system menu.

## System Menu

- `SUPER + L`: lock screen.
- `Sistema`: opens power/session actions.
- `Bluetooth`: opens `bluetui`.
- `Internet`: opens `nmtui`.

## Waybar

- Bluetooth module click: opens `bluetui`.
- Network module click: opens `nmtui`.

## Hyprland TUI Windows

- `bluetooth-tui`: floating, pinned Bluetooth TUI window.
- `wifi-tui`: floating, pinned NetworkManager TUI window.
